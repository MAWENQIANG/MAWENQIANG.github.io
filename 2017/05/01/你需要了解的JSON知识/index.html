你需要了解的JSON知识

Json是什么？

  • JSON是一种数据交换模式，它被许都多的系统用于数据之间的交换。
  • JSON独立于编程语言，虽然的JSON（JavaScript Object Notation）的英文感觉是和JavaScript是息息相关的，但是你用自己的编程语言是完全没有问题的。而且你并不需要先学习JavaScript。
  • JSON拥有很好的可移植性。
